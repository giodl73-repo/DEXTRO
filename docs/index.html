The fairest redistricting method is the oldest one: divide it in half. Split each state's census tracts into two equal populations. Split each half again. Repeat until you have the right number of districts. Edge weights on the graph equal shared boundary lengths, so minimizing the cut automatically minimizes district perimeter — compactness without ever optimizing for it. No political or racial data enters the algorithm at any stage.
