\section{Evaluation}
\label{sec:evaluation}

\subsection{Experimental Setup}
\label{sec:eval:setup}

We implement PFR using the \texttt{redist} Rust binary with $k$-way METIS
partitioning at each tree level. Population balance tolerance $\varepsilon =
0.005$ (0.5\%) is applied at each level. The content-derived seed
$\text{SHA256}(\text{census\_release\_id} \| \text{DIA\_SEED\_V1})$ is used
throughout, ensuring reproducibility without arbitrary seed choice.

We evaluate on all 50 states at their 2020 apportionment seat counts.

\subsection{California: Full Seat-Count Trajectory}
\label{sec:eval:california}

California's seat count has varied from 30 (1950) to 53 (2010) to 52 (2020).
Figure~\ref{fig:california-trajectory} shows the factorization structure and
map at each count from 48 to 53.

The $51 \to 52$ transition:
\begin{itemize}
  \item $51 = 3 \times 17$: trisect state (NorCal / Central / SoCal), then
        17-cut each third.
  \item $52 = 4 \times 13 = 2^2 \times 13$: bisect state, bisect each half,
        then 13-cut each quarter.
\end{itemize}
The top-level structure changes from 3-way to 2-way. No geographic splits are
reused. This is a complete structural break forced by arithmetic.

The $52 \to 53$ transition ($53$ is prime):
\begin{itemize}
  \item $53$: single 53-way cut of the whole state. No reuse of the 52-seat
        factorization tree at any level.
\end{itemize}

Despite the structural breaks, the partisan outcomes (Dem seat share) remain
stable across the trajectory. % TODO: compute and report

\subsection{Wisconsin: Powers-of-2 Stability}
\label{sec:eval:wisconsin}

Wisconsin ($n = 8 = 2^3$) has a fully binary factorization tree. Its PFR map
is identical to the MEC map produced by recursive bisection in
\citet{dellaLibera2026mec}: 3D/5R ($-12.8$\,pp).

Going from 8 to 9 ($= 3^2$) would require discarding the binary tree and
replacing it with a 2-level ternary tree. This is a full structural break.
Going from 7 (prime) to 8 ($2^3$) would replace a 7-cut with a binary tree.

\subsection{Pennsylvania: Prime Seat Counts}

\subsection{Comparison to MEC}
\label{sec:eval:comparison}

For states with $n = 2^k$ (pure powers of 2), PFR and MEC are identical by
construction. For other states, PFR may differ because:
\begin{itemize}
  \item MEC uses repeated bisection (always 2-way splits), regardless of $n$.
  \item PFR uses the factorization-prescribed $k$-way splits, which may find
        different minimum-cut boundaries than iterated bisections.
\end{itemize}
