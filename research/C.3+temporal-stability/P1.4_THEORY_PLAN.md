## Section 3.6 Structure

### 3.6.1: Spectral Stability of Hierarchical Partitioning
**Key Idea**: Binary splits follow Fiedler vector (2nd eigenvector of graph Laplacian), which evolves smoothly under graph perturbations

**Theory to Cite**:
- Fiedler (1973): Algebraic connectivity and spectral bisection
- Pothen et al. (1990): Spectral partitioning of graphs
- Chung (1997): Spectral Graph Theory - perturbation bounds

**Our Argument**:
```
Graph G_t evolves smoothly from 2010 to 2020:
- Edge weights change gradually (demographics shift incrementally)
- Graph structure remains similar (adjacency mostly preserved)

Fiedler vector v₂(G_t) is continuous function of edge weights
→ v₂(G_2010) ≈ v₂(G_2020) for smooth graph evolution
→ Binary splits based on v₂ remain similar
→ Top-level structure (Level 0) perfectly stable
```

**Support**: Compute Fiedler vector similarity for Alabama/Georgia (dot product)

### 3.6.2: Modularity and Hierarchical Structure
**Key Idea**: Top-level splits maximize modularity Q, which is robust to local perturbations

**Theory to Cite**:
- Newman & Girvan (2004): Finding and evaluating community structure
- Fortunato (2010): Community detection in graphs (review)
- Clauset et al. (2004): Hierarchical structure and modularity

**Our Argument**:
```
Recursive bisection optimizes modularity at each level:
- Level 0: Maximize Q for 2-way partition (strongest division)
- Level 1: Maximize Q within each subtree
- Level 2+: Fine-grained optimization

High modularity Q ⇔ strong community structure
→ Stable under perturbations (adding/removing edges)
→ Top-level splits (highest Q) most stable
→ Bottom-level splits (lower Q) less stable
```

**Support**: Compute modularity Q for each tree level (2010 vs 2020)

### 3.6.3: Optimization Landscape: Binary vs K-Way
**Key Idea**: Binary partitioning has unique global optimum (Fiedler), k-way has many local optima

**Theory to Cite**:
- Hendrickson & Leland (1995): Multilevel recursive bisection
- Karypis & Kumar (1998): METIS algorithms and optimization landscape
- Buluç et al. (2016): Recent advances in graph partitioning

**Our Argument**:
```
Binary split (k=2):
- Optimal partition = Fiedler vector sign
- Unique global optimum (modulo tie-breaking)
- Deterministic for same input

K-way split (k≥3):
- NP-hard optimization problem
- Many local optima (exponentially many for large k)
- Random seed affects outcome
- METIS uses multilevel coarsening + refinement (heuristic)

Temporal stability:
- Binary: Same input graph → same Fiedler vector → same split
- K-way: Input changes slightly → may jump to different local optimum
```

**Support**: Literature review (no computation needed)

### 3.6.4: Perturbation Analysis
**Key Idea**: Hierarchical methods are more robust to edge weight perturbations

**Theory to Cite**:
- Davis & Kahan (1970): Perturbation of eigenspaces
- Stewart & Sun (1990): Matrix Perturbation Theory
- Von Luxburg (2007): Tutorial on spectral clustering

**Our Argument**:
```
Let G₀ = 2010 graph, G₁ = 2020 graph
Perturbation: ΔG = G₁ - G₀ (edge weight changes)

For hierarchical bisection:
||P_H(G₁) - P_H(G₀)|| ≤ f(||ΔG||, depth)

where f increases with depth:
- Level 0: Very robust (largest eigenvalue gap)
- Level 1: Moderately robust
- Level 2+: Less robust (smaller eigenvalue gaps)

For k-way:
||P_K(G₁) - P_K(G₀)|| can be large even for small ||ΔG||
(due to discrete optimization jumping between local optima)

Empirical validation:
- 100% structural stability (Level 0-2) confirms theory
- 71.6% assignment stability (deep levels + census redrawing)
```

**Support**: Informal argument + empirical validation (no heavy math)

### 3.6.5: Connection to Empirical Findings
**Key Idea**: Theory predicts exactly what we observed

**Connections**:
1. **100% structural stability** (Section 3.5)
   - Theory: Fiedler vector stability → binary split stability
   - Evidence: All binary paths identical 2010→2020

2. **Level-wise stability gradient** (not measured yet, but predictable)
   - Theory: Eigenvalue gap decreases with depth → less stability
